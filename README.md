This project was made for Resolution Coding Week 4. It includes two C programs: a calculator and a word counter.

- Calculator

The calculator asks the user for two numbers and an operation (`+`, `-`, `*`, or `/`) and then prints the answer.

- What I learned
This project helped me learn:
- How to write basic C programs.
- How to use `printf` and `scanf`.
- How to use variables and types in C.
- How to use loops and if statements.
- How to read files in C.
- How to compile programs with GCC.

- Notes

On Windows, I used VS Code and GCC/MinGW to compile the programs.
